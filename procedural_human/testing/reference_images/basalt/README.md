# Basalt Column Reference Images

Place reference images of Icelandic basalt columns here for visual comparison during validation.

## Recommended Sources

1. **Stuðlagil Canyon** - Tall vertical columns with blue glacial water
   - Search: "Stuðlagil Canyon basalt columns"
   - Features: 20-30 meter columns, dramatic vertical orientation

2. **Reynisfjara Beach** - Coastal basalt near Vik
   - Search: "Reynisfjara beach basalt columns Iceland"
   - Features: Weathered tops, sea spray erosion, cave formations

3. **Svartifoss Waterfall** - Basalt columns behind waterfall
   - Search: "Svartifoss basalt columns"
   - Features: Dark columns, hanging column tips

## Key Visual Features to Match

- **Hexagonal cross-section** (5-7 sides, predominantly 6)
- **Vertical orientation** with slight irregularity
- **Stepped/varied heights** between adjacent columns
- **Visible gaps/cracks** between columns
- **Flat or slightly weathered tops**

## Naming Convention

- `stuðlagil_01.jpg` - Stuðlagil Canyon reference
- `reynisfjara_01.jpg` - Reynisfjara Beach reference
- `svartifoss_01.jpg` - Svartifoss reference

## Usage

These images are used by the validation system to compare rendered outputs
from the `BasaltColumns` geometry node group against real-world examples.
